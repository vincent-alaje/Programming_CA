Digital diary 
pre-requirements
Laptop should have on it installed:
Python 3.9.4(https://www.python.org/downloads/)
Django
VS Code(https://code.visualstudio.com/download)
Step 1: download python latest version
Step 2: download VS code latest version
Step 3: open cmd and write command (pip install django)

The application is built with python and Django. To run or build the app yourself, you'll need to have Django framework in VS code or visual studio code and python 3.9.4. installed.
Running the app
Install the pre-requirements
Navigate into the project directory: cd Digital_Diary
Extract the zip folder



Open the VS Code
-	Click on open folder option
-	Select the folder containing the code
-	Click on Terminal button
-	Click on new Terminal
-	Run the command "cd digitaldiary" (No need to run this if you are already in this directory)
-	Run the command “python manage.py makemigrations”
-	Run the command “python manage.py migrate”
-	Run the command "python manage.py runserver"
-	You'll get a link. Click on that (http://127.0.0.1:8000/)
-	You'll be redirected to the Digital Diary website.
